# 🔍 GitHub Profile Analyzer

A Python command-line tool that analyzes any GitHub profile and gives personalized recommendations to help developers improve their presence.

## 🔗 Live Demo
Try it yourself — enter any GitHub username!

## ✨ Features
- 👤 **Profile Analysis** — View followers, repos, bio and location
- 🏆 **Profile Score** — Get scored out of 100 based on your GitHub presence
- 📈 **Commit Activity** — See how active you are on GitHub
- 🧑‍💻 **Language Breakdown** — See your most used programming languages
- ⭐ **Top Repositories** — View your most starred projects
- 💡 **Recommendations** — Get personalized tips to improve your profile
- ⚔️ **Compare Profiles** — Compare two GitHub profiles side by side
- 🔍 **Search by Language** — Find top developers by programming language
- 📄 **PDF Export** — Export your analysis as a professional PDF report

## 🚀 How to Run

**1. Clone the repo**
```bash
git clone https://github.com/PushpaSaysHi/github-analyzer.git
cd github-analyzer
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Run the app**
```bash
python main.py
```
